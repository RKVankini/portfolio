Weather Dashboard (Frontend Project)

A clean, client-side weather application built with JavaScript, powered by the OpenWeather API and deployed on GitHub Pages.

Project Overview

The RK Weather Dashboard is a lightweight frontend application designed to deliver real-time weather information with a strong focus on simplicity, performance, and user experience. The project runs entirely in the browser without any backend services and is optimized for static hosting.

Key Features & Responsibilities

  • Integrated OpenWeather API to fetch real-time weather data.
  • Implemented city-based weather search with validation and error handling.
  • Displayed temperature, humidity, wind speed, and weather conditions.
  • Built a Light / Dark theme toggle with persistent user preference using localStorage.
  • Handled API and network errors gracefully using inline UI messages.
  • Ensured responsive layout for both desktop and mobile devices.
  • Structured configuration using env.js and config.js for maintainability.
  • Deployed the application using GitHub Pages for fast global access.

Tech Stack

  • HTML5, CSS3
  • JavaScript (ES6)
  • OpenWeather API
  • GitHub Pages (Deployment)

Design & Engineering Decisions

  • Kept the project client-side to match the needs of a static portfolio app.
  • Avoided unnecessary frameworks or build tools for clarity and simplicity.
  • Focused on UX, readability, and maintainability over visual clutter.
  • Documented API key visibility honestly as a frontend limitation.
