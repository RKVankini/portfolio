DevOps Web App Deployment (AWS)

Scalable, containerized web application deployed using AWS services, Docker, Kubernetes (Minikube for POC), and automated provisioning with Ansible.

Project Overview

Deployed frontend (Apache) and backend (Node.js + PM2) on EC2 instances inside a custom VPC. Backend connects to an RDS MySQL instance. Application was containerized using Docker and tested on Kubernetes (Minikube) for orchestration proof-of-concept.

Key Responsibilities

  • Created EC2 instances and configured networking (VPC, subnets, security groups).
  • Configured RDS MySQL and connected the backend securely using AWS Secrets Manager.
  • Wrote Dockerfiles and Docker Compose for local development and container testing.
  • Tested orchestration using Minikube and kubectl (Deployments, Services).
  • Automated server provisioning tasks using Ansible playbooks (package install, service start, environment setup).
  • Maintained code and deployment instructions in a GitHub repository with clear README.

Tech Stack

  • Cloud: AWS (EC2, RDS, Route 53)
  • Containers: Docker, Kubernetes (Minikube)
  • Automation: Ansible, Bash
  • App: Node.js, Apache
  • CI / Repo: GitHub

Outcome

Delivered a reproducible deployment process, improved development-to-deployment parity via containers, and automated provisioning to reduce manual setup time.
