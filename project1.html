DevOps Web App Deployment (AWS)

Scalable, containerized web application deployed using AWS services, Docker, Kubernetes (POC), and Ansible.

Project Overview

Deployed frontend (Apache) and backend (Node.js + PM2) on EC2 inside a custom VPC. Backend connects to RDS MySQL. Application containerized with Docker and tested on Minikube for orchestration POC.

Key Responsibilities

  • Provisioned EC2 instances and configured VPC networking and security groups.
  • Configured RDS MySQL and used AWS Secrets Manager for credentials.
  • Created Dockerfiles and Docker Compose for development and testing.
  • Tested Kubernetes orchestration using Minikube (Deployments, Services).
  • Wrote Ansible playbooks for server setup and service configuration.
  • Maintained GitHub repo with README and deployment steps.

Tech Stack

  • AWS: EC2, RDS, Route53
  • Containers: Docker, Kubernetes (Minikube)
  • Automation: Ansible, Bash
  • App: Node.js, Apache
  • Version Control: GitHub
