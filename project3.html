AI Automation & Integration

Prototype connecting ML model inference to dashboards and automated pipelines for insights and reporting.

Project Overview

Built a prototype pipeline that collects data, preprocesses it, trains a lightweight model, and serves predictions through an API. Results are visualized in a dashboard and delivery is automated via scripts and containerized services.

Tech Stack

  • Frontend: Vue.js
  • Backend / API: Python (FastAPI)
  • ML: TensorFlow / scikit-learn
  • DB: MySQL / MongoDB
  • Containers: Docker
  • Visualization: Chart libraries / Dashboards
