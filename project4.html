Mini Game Hub — Browser-Based Games Platform (HTML, CSS & JavaScript)

A lightweight, interactive mini game hub featuring multiple browser-based games with modular architecture, theme switching, sound effects, and responsive UI — built using pure frontend technologies.

Project Overview

The Mini Game Hub is a frontend-focused project designed to showcase strong JavaScript fundamentals, clean UI design, and scalable code organization. It contains multiple mini games accessible from a unified hub interface, allowing users to play instantly without any backend or external libraries.

The project emphasizes modular game loading, reusable components, audio feedback, and theme persistence (dark/light mode), making it easy to extend with additional games in the future.

Application Architecture

Core Modules & Flow

  1. Home / Hub Screen: Displays available games with clickable cards.
  2. Game Loader: Dynamically loads selected game logic and UI.
  3. Game Engine: Handles gameplay logic (timers, scoring, win/loss conditions).
  4. Sound Engine: Plays feedback sounds (click, correct, wrong, timeout, win).
  5. Theme Manager: Toggles and persists light/dark mode using local storage.
  6. Responsive UI: Ensures usability across desktop and mobile devices.

Games Included

  • Memory Card Game
  • Quiz Game
  • Reaction / Logic-based Mini Games
  • Expandable structure for future games

Tools & Tech

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • UI/UX: Responsive layout, theme toggle (dark/light)
  • Audio: Native JavaScript Audio API
  • Version Control: Git & GitHub
  • Deployment: GitHub Pages

Key Responsibilities & Achievements

  • Designed a modular frontend architecture allowing easy addition of new games.
  • Implemented a centralized sound engine for consistent audio feedback.
  • Built theme toggle functionality with persistent user preference.
  • Ensured responsive design for both desktop and mobile gameplay.
  • Maintained clean folder structure and readable, maintainable JavaScript code.
  • Deployed the complete project as a static site using GitHub Pages.

How to Run the Project

  1. Clone the repository from GitHub.
  2. Open index.html in a browser or serve using a local static server.
  3. Select a game from the hub and start playing instantly.
